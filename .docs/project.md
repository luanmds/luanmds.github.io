# project.md

## O que é

Blog pessoal de Luan Mello — engenheiro de software back-end do Rio de Janeiro, especializado em .NET e arquitetura distribuída. O blog é um espaço de **notas públicas**: aprendizados documentados para si mesmo que, por serem públicos, podem ser úteis para outros.

## Propósito

Documentar aprendizados técnicos durante ciclos de exploração (um novo projeto, uma tecnologia sendo estudada, uma decisão de arquitetura tomada). Não há compromisso de calendário — a publicação acontece quando há algo relevante a registrar.

## Problema que resolve

A perda de conhecimento tácito. Soluções encontradas, decisões avaliadas e experimentos feitos tendem a se perder sem um espaço dedicado de registro. O blog serve como memória externa técnica do autor.

## Domínio

Não há domínio de negócio. O projeto é um gerador de site estático (SSG) — não existe lógica de aplicação, estado compartilhado ou regras de negócio. O "domínio" é editorial: criação e publicação de conteúdo Markdown.

## Audiência

O autor. Leitores externos são bem-vindos, mas não são o critério primário de decisão de conteúdo.
