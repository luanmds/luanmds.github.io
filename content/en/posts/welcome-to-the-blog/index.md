## Hello, world! 👋

This is the first post on my blog. I decided to create a space to share articles about **software development**, tools, experiences and whatever I find relevant along the journey.

## How the blog was built

The blog was built with the following technologies:

- **[Hugo](https://gohugo.io/)** — blazing fast static site generator
- **[PaperMod](https://github.com/adityatelange/hugo-PaperMod)** — minimalist theme with dark mode and search support
- **[GitHub Pages](https://pages.github.com/)** — free hosting
- **[GitHub Actions](https://github.com/features/actions)** — CI/CD for automatic builds and deployments

All content is written in **Markdown** and organized as [Page Bundles](https://gohugo.io/content-management/page-bundles/), keeping images co-located with each article.

## Languages

The blog is bilingual: articles will be written in **Portuguese (default)** and **English**.

You can switch languages using the selector in the menu.
